# 🛒 Amazon UI Clone

This is a responsive **Amazon.com homepage UI clone** built using **HTML**, **CSS**, and **Font Awesome** icons. It replicates the look and feel of Amazon's website including the navbar, search bar, shopping sections, and footer.

---

## 🔥 Live Demo

You can open `index.html` in your browser or use a live server to see the project in action.


---

## 💡 Features

- ✅ Amazon-style top navbar with location and cart icon
- 🔍 Functional-looking search bar
- 📦 "Deals" section with images and offers
- 🎨 Styled using **pure CSS**
- 📱 Responsive design with `flexbox`
- 🎉 Font Awesome icons for enhanced UI

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **Font Awesome** CDN
